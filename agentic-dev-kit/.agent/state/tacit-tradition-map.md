# Tacit Tradition Map（隐性传统地图）

## 如何使用

- 在 `/review` 的 Expert A 阶段，参考本文件中的传统约定作为审查标准的补充。
- 在 `/evolve` 过程中，当新规约与隐性传统冲突时，需在此文件中记录解决方案。
- 在 `autoresearch:review` 中，将本文件作为 Polanyi Protocol 的落盘记忆。
